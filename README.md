# Учебный проект Дмитрия С.

Простой и аккуратный статический сайт, созданный с помощью HTML, CSS и JavaScript. Проект включает несколько страниц, адаптивную верстку и клиентскую логику для улучшения взаимодействия с пользователями.

## Структура проекта

```text
Dmitriy_project_HTML/
├── .gitignore          # Файлы, игнорируемые Git
├── css/
│   └── style.css       # Основные стили сайта
├── js/
│   └── script.js       # Скрипты для интерактива и валидации форм
├── img/                # Изображения и графические ресурсы
├── index.html          # Главная страница
├── contacts.html       # Страница контактов
├── services.html       # Страница услуг
└── README.md           # Описание проекта
```

## Технологии

* **HTML5** — семантическая разметка для удобства чтения и SEO.
* **CSS3** — адаптивная стилизация, Flexbox/Grid и CSS-переменные.
* **JavaScript** — работа с DOM, обработка событий и простая валидация форм.

## Особенности проекта

* Чистая структура файлов и понятная навигация.
* Адаптивный дизайн для разных экранов.
* Интерактивная форма обратной связи на странице `contacts.html`.
* Использование современных HTML/CSS практик.

## Как открыть проект

1. Откройте папку `Dmitriy_project_HTML`.
2. Запустите файл `index.html` в браузере.

> Проект не требует сборки и установленного сервера — достаточно открыть HTML-файлы напрямую.
