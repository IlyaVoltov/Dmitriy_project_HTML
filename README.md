```text
## Технологический стек
* **HTML5:** Семантическая разметка (использование тегов `<header>`, `<main>`, `<footer>` для SEO и доступности).
* **CSS3:** * Методология Flexbox и Grid для построения сеток.
    * Использование CSS-переменных (`:root`) для легкого брендирования.
    * Адаптивная верстка (использование `max-width` и гибких единиц измерения).
* **JavaScript (Vanilla):** Обработка событий DOM, валидация форм и создание интерактивных уведомлений.

## Основные возможности
1. **Навигационная система:** Реализовано сквозное меню на 9 разделов согласно ТЗ.
2. **Форма записи:** Интерактивная форма с проверкой обязательных полей и обратной связью через JS.
3. **Графический контент:** Логотип является активной ссылкой на главную страницу (Задание №1, п.7).
4. **Документирование:** Код снабжен подробными комментариями (Задание №2, п.10).
```
